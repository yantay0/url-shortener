# Go URL Shortener

## Overview
This is a simple URL shortener written in Go. It allows you to shorten long URLs into manageable links that never expire.

## Features
- Shorten URLs
- Custom alias for URLs
- Basic analytics (click counts)

## Getting Started

### Prerequisites
- Go 1.16 or higher
- PostgreSQL 13.0 or higher
